Graphical Content Recognition with Editable Charts and Layout Retention

The task of extracting and processing structured content from textbooks and documents containing textual and visual elements such as charts and graphs requires a advanced tool capable of detection, localization, processing and reconstruction. Although tools utilizing modern OCR technologies can extract textual contents with high accuracy and speed, there is a lack of a tool that can allow for processing and editing of the detected visual elements. To address this challenge, we present a serial pipelining structure that integrates Detectron2 with Faster R-CNN model, an advanced pre trained object detection model to extract document content and visual features alongside PaddleOCR, a state of art OCR model. In our system, all the textual content is extracted using PaddleOCR while custom algorithms like k-means clustering, contour analysis and spatial reasoning allows for complete feature extraction like bar heights, pie slices and reconstruction of visual elements. Our key achievements includes techniques to develop customized dataset for training, adaptive label mappings to dynamically associate extracted data with its corresponding visual component and reconstruction techniques. This tool advances document digitization by automatic extraction, interactive GUI to edit the extracted visual elements and output generation while retaining original document layout.
